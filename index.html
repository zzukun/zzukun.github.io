Ubuntu指令与快捷键

发表于 2016-03-22   |  

命令

修改hosts文件后重启网络服务：
sudo /etc/init.d/networking restart

清屏：
clear

删除文件：
rm 文件名 （删除一个文件
rm -r 文件夹名 （删除一个非空文件夹
rm -rf 文件夹名 （强力删除一个非空文件夹，不要求确认

查看内存与CPU的使用情况：
top
输入top命令后可以查看各个进程的PID
假设我们想要关闭某个进程，其PID是123，那么：
sudo kill 123

快捷键

打开一个新终端：
Ctrl + Alt + T
